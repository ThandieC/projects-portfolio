RECIPE BOOK

Tech Stack: JavaScript, Angular, Bootstrap, CSS3, HTML5, Firebase, Git, Github.

  • An app designed for the user to create and keep an editable collection of recipes.
  • The user can also make or edit a list of ingredients for their Shopping from the app and save them in a Firebox database.
  • The App is authenticated so users have to sign up to access the entire App.
